Acerca de Mí

Soy José Alejandro Cujar Barrero, un colombiano de 25 años con habilidades en diseño y publicidad, utilizando principalmente las herramientas gráficas de la Suite de Adobe. Recientemente, he complementado mis habilidades de diseño con el desarrollo FULL STACK, lo que me ha permitido mantenerme a la vanguardia tecnológica y cumplir con las necesidades de mis clientes. Valoro el aprendizaje continuo como clave para el éxito.

Habilidades

Soy un profesional creativo con experiencia en desarrollo web full stack, gestión de redes sociales, diseño gráfico, SEO, SEM y análisis de métricas. Me destaco por mi adaptabilidad, liderazgo, resolución de problemas y orientación al cliente, combinando creatividad e innovación para desarrollar estrategias efectivas y persuasivas.

HTML

Domino la estructura semántica de HTML, la creación de contenido accesible y organizado, la optimización de formularios, imágenes y multimedia, así como el uso de estándares HTML5 para desarrollar sitios modernos y responsivos. Además, integro SEO, aseguro compatibilidad entre navegadores y colaboro con CSS y JavaScript para páginas dinámicas y funcionales.

CSS3

Domino el uso de CSS para crear diseños atractivos y responsivos, aplicando técnicas avanzadas como flexbox, grid, animaciones y transiciones. Optimizo la presentación visual y la experiencia del usuario, asegurando la compatibilidad entre navegadores y el buen rendimiento en diferentes dispositivos.

Javascript

Tengo experiencia en JavaScript para desarrollar aplicaciones interactivas y dinámicas. Domino el uso de funciones, objetos, arrays y manipulaciones del DOM para crear experiencias de usuario fluidas. También manejo la asincronía con promesas y async/await, así como la integración de APIs para mejorar la funcionalidad. Utilizo ES6+ para optimizar el código y trabajar con módulos, y aplico buenas prácticas para garantizar un código limpio y eficiente.

React

Tengo experiencia en React, creando interfaces interactivas con componentes funcionales y hooks. Utilizo Redux y Context API para manejar el estado, React Router para la gestión de rutas y optimizo el rendimiento con técnicas como la renderización condicional y el memoization.
